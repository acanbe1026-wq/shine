## 프로젝트 기획 단계

### 핵심 메시지 정의

1. 프로젝트 개요
- 프로젝트 명 : 계약의 신
- 랜딩페이지의 목적 : 계약의 신 플랫폼 소개, 브랜드 인지도
- 타켓 오디언스 : 공인중개사사무소 대표 및 임직원
- 핵심 포인트 및 솔루션 : 국내 유일 반응형 웹 기반 공인중개사 업무 플랫폼 ERP
- 원하는 행동 : 사용자(공인중개사사무소 대표 및 임직원)가 랜딩페이지를 접속해서, 상담요청 및 계약의 신 가입을 유도

2. 계약의 신 플랫폼 주요기능
- 매물관리, 고객관리, 계약관리, 협력업체관리, 일정관리, 만기관리, 문자관리, 통화관리 간에 상호 알고리즘 적용으로 유기적으로 작동
- ID와 PW만 있으면 집에서도 재택근무 가능(컴퓨터, 스마트폰으로 완벽한 업무실현)
- 내 고객들 만기를 몰라도 자동으로 만기일 database update 구현
- 스마트폰 APP구현으로 고객이 누구인지 알수있고, 통화하면서 업무를 동시에 가능
- 아파트/오피스텔/다가구/다세대/주택/빌딩/상가의 모든 매물관리 할 수 있는 시스템 구현
- 철저한 직원관리고 직원들의 업무실적 및 개인별 매출관리, 업무별 담당자 지정기능으로 직원들간의 비공개업무 가능
- 내가 원하는 조건의 고객들에게 단체문자 발송기능
- 사용자별 설정을 통해 자동업무할당기능으로 매일 중개사가 해야할 업무를 AI할당해주는 기능

### 컨셉 & 테마 결정

- 테마 : "Chaos to Order" (혼돈에서 질서로)
- 비쥬얼 : 다크모드 + 네온색상(사이버펑크 스타일), 그라디언트
- 톤앤매너 : 전문적
- 컬러 스킴 : 신뢰를 상징하는 딥 블루(Deep Blue) 또는 차분한 네이비를 메인 컬러로 사용하고, 강조하고 싶은 CTA 버튼에는 대비되는 오렌지나 그린을 사용
- 폰트 : 가독성이 높은 고댁 계열(pretendard)을 사용
- 데이터 시각화 : 복잡한 기능 설명보다는 아이콘과 인포그래픽을 활용해 직관적으로 정보 전달
- 애니메이션 : GSAP 기반 스크롤 애니메이션
- 스타일 : Tailwind CSS 사용
- 디자인 방향 : 인터랙티브한 요소(호버 효과, 패럴랙스), 부드러운 전환 애니매이션

### 페이지 구조 설계

1. Hero Section(첫인상)
- 큰 헤드라인 : 엑셀, 구글워크시트, 설치형 프로그램 이제 그만 !! 계약의 신과 함께라면 업무효율화와 매출상승의 꿈이 실현됩니다.  
- 서브 카피
- 스크롤 힌트
- 비주얼 : 태블릿이나 모바일에서 플랫폼이 구동되는 깔끔한 인터페이스 이미지를 배치해 전문성 부여

2. Problem Section(문제제기)
- 기존 엑셀이나 구글워크시트만으로는 현재의 대량의 DB를 효과적으로 관리가 어려움
- 기존 타사프로그램은 설치형으로 재택근무가 불가능함
- 현재 공인중개사가 업무를 하는 효율적이고 정확하며 자동화 할수 있는 업무 플랫폼이 부재한 상황
- 진입장벽이 낮은 공인중개사무소특성상 경쟁이 치열하여 차별화 할수 있는 플랫폼이 절실히 요구됨
- 방문, 계약, 잔금 일정, 매물관리, 고객관리, 만기관리 등 많은 일들을 챙겨줄 수 있는 정확한 플랫폼이 요구됨
@ 위 문제제기는 시각화

3. Solution Section(해결책)
- 반응형 웹, APP구현으로 언제 어디서나 대량의 DB를 관리하고 고객을 응대할 수 있음
- 중개사가 매일 해야할 일들을 계약의 신이 할당해줌
- 방문, 계약, 잔금 일정, 매물관리, 고객관리, 만기관리의 내용이 유기적으로 연동되어 꼼꼼하고 똑똑한 업무자동화 실현

4. CTA Section(행동 유도)
- 인터뷰 내용을 카카오톡 채팅 대화방식으로 기재
  (인터뷰 내용)
  a 엑셀, 구글워크시트, 타사 프로그램을 써봤는데, 이거 작품이네요 - 강동구 고덕동 A 공인중개사 대표
  b 스마트폰 앱/PC로 어디서든 업무가 가능해서 너무 좋네요 - 경기도 용인기흥구 P 소속공인중개사
  c 걸려오는 전화 고객정보를 핸드폰에서 즉각 확인해서 고객과 통화하니 marketing에 너무 좋아요 - 강남구 개포동 K공인중개사 대표
  d 우리 중개업소는 건물/아파트/오피스텔/상가/주택을 모두 취급하는데, 이 프로그램은 이 모든것들을 관리할 수 있어서 너무 좋네요 - 인천광역시 남동구 간석동 S공인중개사 대표
  e 어떤 업무도 놓치지 않게, 아침에 출근하면, 해야할 일을 부여해주니 정말 똑똑한 AI비서네요 - 대전광역시 유성구 S 중개보조원 
- 고객이 상담문의를 통해, 플랫폼 사용 및 가입상담을 받을수 있도록 시스템 구현
  (상담문의시 성명, 연락처, E-mail주소, 문의내용을 입력해서, 문의하기 버튼을 누르면, 계약의 신에 알리는 시스템구현)
- 채널톡을 통해 24시간 상담 및 가입을 유도할 수 있는 시스템 구현
